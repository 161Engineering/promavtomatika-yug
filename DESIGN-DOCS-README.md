## 🎯 Быстрый старт

### Создание новой страницы

1. **Скопируйте базовую структуру** из `COMPONENTS-REFERENCE.md`
2. **Используйте section-header** для каждой секции с контентом
3. **Выберите нужный тип карточек** из справочника
4. **Проверьте соответствие** по чек-листу из `DESIGN-SYSTEM.md`
5. **Сверьтесь с примерами** "Правильно vs Неправильно" из `PATTERNS-GUIDE.md`

### Проверка существующей страницы

1. Откройте `PATTERNS-GUIDE.md`
2. Проверьте наличие типичных ошибок
3. Используйте чек-лист из `DESIGN-SYSTEM.md`
4. Исправьте несоответствия согласно паттернам

## 🔑 Ключевые принципы

### 1. Container + Grid вложенность
```html
✅ ПРАВИЛЬНО:
<div class="container">
    <div class="competency-grid">
        <!-- карточки -->
    </div>
</div>

❌ НЕПРАВИЛЬНО:
<div class="container competency-grid">
    <!-- карточки -->
</div>
```

### 2. Section-header обязателен
```html
<section class="section">
    <div class="container">
        <div class="section-header">
            <div>
                <span class="section-tag">Метка</span>
                <h2>Заголовок</h2>
            </div>
            <p class="section-note">Пояснение</p>
        </div>
        <!-- Контент секции -->
    </div>
</section>
```

### 3. Изображения полностью атрибутированы
```html
<!-- IMAGE PROMPT: описание сцены -->
<div class="project-photo-card">
    <img src="images/name.jpg" 
         alt="Детальное описание"
         width="1120" height="720" 
         loading="lazy" decoding="async">
</div>
```

### 4. Стандартные классы кнопок
```html
<a class="button button-primary" href="#">Главная кнопка</a>
<a class="button button-secondary" href="#">Вторичная</a>
```

### 5. Иконки в icon-badge
```html
<span class="icon-badge" aria-hidden="true">
    <svg viewBox="0 0 24 24"><!-- SVG --></svg>
</span>
```

## 📏 Border Radius

- **10px** — мелкие элементы
- **12px** — средние кнопки
- **16px** — фото, KPI-блоки
- **22px** — карточки, панели (var(--radius))
- **999px** — pill-элементы (теги)

## ✅ Чек-лист качества

При создании/проверке страницы:

- [ ] Header с правильными классами
- [ ] Footer с footer-grid
- [ ] Все секции в `<section class="section">`
- [ ] Grid-классы вложены в `.container`
- [ ] Section-header с section-tag + h2 + section-note
- [ ] Изображения с IMAGE PROMPT комментарием
- [ ] Width, height, alt у всех изображений
- [ ] Кнопки используют button-классы
- [ ] Карточки используют стандартные классы
- [ ] Icon-badge для SVG иконок
- [ ] Цвета соответствуют палитре
- [ ] Border-radius системный
- [ ] Hover-эффекты работают
- [ ] Aria-атрибуты где необходимо

### Скрипт исправления Grid-структуры
Если обнаружены проблемы с grid-контейнерами:

